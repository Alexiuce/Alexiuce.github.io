最近使用GitHub的时候,经常感觉速度越来越慢,刚刚看到网上的一个技术头条,试着用下,效果还真不错,一下子腰也不疼了…..

Git 加速步骤

一. 打开终端后输入下面命令:

  • 默认情况下config这个文件是没有的,如果没有,命令会建立config文件,如果文件存在,则打开文件.
  • 以.开头的 文件或文件夹表示隐藏,需要使用ls -a命令才能查看到.

二. 编辑文件内容

  • ControlPath /tmp/%r@%h:%p: %r代表远程登录用户名，一般都为git，%h表示目标主机，%p表示端口。
  • ControlPersist yes: 持久保持链接,意思就像打电话,即便双方不说话,但也不要挂断.
  • ControlMaster auto: 共享一个已经存在的连接，如果没有，则自动创建一个连接。

三. 保存退出vim编辑器

四. 开始Git吧
